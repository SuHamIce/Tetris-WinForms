# 俄罗斯方块 (Tetris) - V3.0 极简暗黑重置版 🎮

本项目是一款基于 C# Windows Forms 开发的现代化俄罗斯方块游戏。
在保留经典核心玩法的基础上，V3.0 版本对视觉 UI 进行了全面重构，致力于提供“街机级”的沉浸式游玩体验。

## ✨ V3.0 核心特性 (Features)

* **纯净暗黑视效 (Dark Mode UI)**：摒弃冗杂的背景，采用纯正的碳黑背景底板，搭配高对比度的“霓虹发光”色块，大幅降低长时间游玩的视觉疲劳。
* **全息落点预测 (Ghost Block)**：新增半透明的幽灵方块投影，精准提示方块的最终落点，极大提升游戏策略性。

## ⌨️ 游戏操作指南 (Controls)

本游戏支持全键盘流畅操作：

* `←` / `→` : 左右移动方块
* `↑` : 旋转方块
* `↓` : 加速下落 (软降)
* `Space (空格键)` : 瞬间触底 (硬降)，高手必备！
* `Enter (回车键)` : 游戏结束后，一键快速重新开始

## 🚀 如何运行 (How to Run)

本程序为绿色免安装版，追求极致轻量化。
1. 解压收到的压缩包。
2. 进入 `Release` 文件夹。
3. 双击运行 `Tetris.exe` 即可开始游戏。

## 🛠️ 技术栈 (Tech Stack)

* **语言**: C# (面向对象设计)
* **框架**: .NET Framework / Windows Forms
* **渲染**: GDI+ (双缓冲防闪烁绘图技术)
