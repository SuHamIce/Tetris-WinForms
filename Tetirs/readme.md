# Tetris-WinForms (俄罗斯方块 2.0)

基于 C# WinForms 开发的经典俄罗斯方块游戏。
在 1.0 基础版本之上，本项目进行了底层核心逻辑的大规模重构，致力于还原现代商业级俄罗斯方块（Tetris Guideline）的丝滑操作手感与核心机制。

## ✨ 核心特性与技术亮点 (v2.0 新增)

* **现代方块生成机制 (7-Bag Randomizer)**
    彻底摒弃了传统的纯随机生成，采用业界标准的 7-Bag 算法。确保每 7 个方块必然包含所有 7 种基础形状，既保持了随机性，又避免了“连续不出长条”的极端非酋体验。
* **幽灵方块投影 (Ghost Piece)**
    新增底部落点投影功能。通过逻辑克隆与虚拟掉落算法，实时计算当前方块的最终触底位置，并以半透明边框渲染，极大提升了高倍速下的操作容错率。
* **定制化旋转系统 (Custom Rotation System)**
    针对不同形态的方块重构了旋转算法，告别单一的数学矩阵旋转：
    * `I / S / Z 型`：采用双状态硬编码切换，彻底解决了 4x4 矩阵偶数中心点导致的“长条旋转摇摆 (Wobble)”问题。
    * `T / L / J 型`：保留中心点矩阵公式旋转，维持紧凑手感。
    * `O 型`：剥夺旋转计算，优化性能。
* **智能踢墙与防漏洞机制 (Wall-Kick & Anti-Exploit)**
    * **极限踢墙**：针对 I 型长条贴墙旋转被卡死的问题，引入了 `X ± 2` 的两格平移判定。
    * **封锁爬墙 Bug**：严格限制了向上位移（踢地板）的条件（必须触底且脚下有实体），彻底修复了无限按旋转键导致方块“左脚踩右脚升天”的业界经典漏洞。
* **UI 与渲染优化**
    * 硬降落（空格键）后的 Timer 强制重置，修复了时间戳脱节导致的“下一方块早产”掉落问题。
    * 动态包围盒算法（Bounding Box）：不论 UI 小窗口如何缩放，预览方块永远保持绝对居中渲染。

## 🎮 操作说明

* **← / →** : 左右移动
* **↓** : 加速下落
* **↑** : 顺时针旋转
* **Space (空格)** : 硬降落 (瞬间触底并锁定)
